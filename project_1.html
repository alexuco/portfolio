Case Study: Hospital Vall d'Hebron

Manager Tool app for Hospital Bioinformatics Research centre.

Hospital Vall d'Hebron

  • Industry: Health Care
  • Size: 10000+
  • Website: uebresults.vhir.org
App for dealing with huge amount of data projects requested for both private and public researchers.

Project Requirements

  • High security standards. Meaning, strong user login and safe environment.
  • Automation process. Bioinformatics technician upload the project files to the server and assign them to the customer.
  • UI Admin panel with dashboard for IT and leads, UI Bioinformatics and UI Customers.

Project Overview

Hospital bioinformatician are working whith huge amount of data for each project requested, sometimes several GB each, which was dificult to send to the Customers, unless via pendrive or specific ftp bridges, what meant poor security and zero version control of files.

The Challenge

To have everything on the same app. In one hand follow Hospital's security requirements with app on their own server, strong login, log every person since it's logged, both hospital technician and customers. And in the other hand make a fresh and easy to use App with lots of automation processes.

The Approach & Solution

Designed & developed from scratch to fit 100% of requirements. 3x UI: IT's team; Bio-informatician team and Customer/clients. Every single page is under strong control of who can create, update, delete files, wwho can get them and download locally. For all that said, it's a must to develop Admin panel with 'user Manager', 'notification manager', 'mailing manager', 'log manager', 'ip's manager', 'Backup manager', 'statistics' and much, much more...

Deploy and app on own physical hospital's servers was a bit knightmare because of its extremely cautious enviroment in order to preserve patient's health data.
